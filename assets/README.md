# Turnauksen logo

Lisää virallinen PNG-logo tähän polkuun:

- `assets/teepee-darts-cup-logo.png`

Sovellus näyttää logon automaattisesti sekä aloitusnäkymässä että yläpalkissa.
Jos tiedostoa ei ole, sovellus näyttää tekstipohjaisen varalogon.


## Voittomusiikki

Jos haluat soittaa voittomusiikin ottelun päättyessä, lisää tähän kansioon tiedosto:

- `assets/victory-music.mp3`

Musiikki käynnistyy automaattisesti, kun jompikumpi pelaaja voittaa ottelun (best of 5, eli 3 legiä).
